# minsdl2js
Pure JS SDL2 bindings for NodeJS using ffi-napi
## Supported Libraries
 - SDL2 <br />
 - SDL2_mixer <br />
 - SDL2_ttf <br />
 - SDL2_image <br />
 - SDL2_gfx <br />
 - SDL2_net <br />
 - SDL2_gpu <br />
 - SDL2_sound <br />
 - SDL2_rtf
## Installation
```sh
npm i minsdl2js
```
